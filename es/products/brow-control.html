Brow Control System

Un sistema de cuatro componentes que define cejas arquitectónicas. Micro-lápiz para precisión, pomada para forma, gel para control, extensiones para densidad. Las cejas definen el rostro. Blushless las define con intención.

Componentes: 4 partes
Fin: Arquitectónico
Para: Máxima presencia

Los Cuatro Componentes

Micro Pencil

Lápiz ultra fino para trazos precisos que definen la arquitectura de la ceja. Cada pelo cuenta. Presión superficial para naturalidad, presión profunda para definición arquitectónica.

Pomada Control

Pomada con textura que sostiene y define la forma de la ceja. Cubre espacios sin parecer pintada. Define la línea del arco y la cola sin compromiso.

Brow Gel

Gel de control que mantiene las cejas en su lugar durante todo el día. Sin apelmazamiento. Sin brillo. Control arquitectónico puro.

Brow Extensions

Fibras finas que se adhieren a las cejas existentes para densidad y forma. Aplica densidad donde la necesitas. Define sin ser obvia.
